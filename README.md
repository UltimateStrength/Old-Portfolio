# 🗂️ Old Portfolio

Primeiro portfólio pessoal, desenvolvido como projeto prático durante um curso da **Alura**. Página única com apresentação, seção de projetos e links de contato.

🔗 **[Ver o site](https://marcosulti.github.io/porte-de-oleo/)**

> Este repositório é um fork da conta acadêmica do curso, preservado para não perder o histórico caso a conta original seja removida.

## 📄 Conteúdo

O portfólio é uma single-page com três seções:

- **Apresentação** — foto de perfil e texto com animação de digitação ("Full Stack Brasileiro 🇧🇷")
- **Meus Jogos** — card com banner linkando para os projetos de jogos no [itch.io](https://ultimate-strength.itch.io/)
- **Contato** — links para Twitter, GitHub e Instagram

## 🛠️ Tecnologias

- HTML5 / CSS3
- JavaScript (vanilla) — animação de texto no hero
- Deploy via GitHub Pages

## 📁 Estrutura

```
Old-Portfolio/
├── index.html    # Estrutura da página
├── style.css     # Estilização
├── main.js       # Animação do texto de apresentação
└── assets/       # Foto de perfil, banner, logos de redes sociais, bandeira
```

## 📌 Contexto

Portfolio simples feito como entrega do curso da Alura — o foco era colocar algo no ar rapidamente. A intenção sempre foi refazer do zero num momento mais sério, o que explica o nome "Old Portfolio".
